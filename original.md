```uml
@startuml

opt 未検索
ユーザー -> Webシステム: 商品検索
Webシステム -> DBシステム: 商品検索処理
DBシステム -> DBシステム: 商品検索処理
DBシステム --> Webシステム: 商品検索結果
Webシステム --> ユーザー: 商品検索結果


alt 検索成功
Webシステム -> ユーザー: 検索結果を表示
else 検索失敗
Webシステム -> ユーザー: 検索失敗メッセージを表示

end
end


@startuml

opt 未購入
ユーザー -> Webシステム: 商品購入
Webシステム -> DBシステム: 商品購入処理
DBシステム -> DBシステム: 商品購入処理
DBシステム --> Webシステム: 商品購入結果
Webシステム --> ユーザー: 商品購入結果


alt 購入処理成功
Webシステム -> ユーザー: 購入完了メッセージを表示
else 購入処理失敗
Webシステム -> ユーザー: 購入失敗メッセージを表示

end


@enduml
```
